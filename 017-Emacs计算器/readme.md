#Emacs计算器

##题目描述
　　Emacs号称神的编辑器，它自带了一个计算器。与其他计算器不同，它是基于后缀表达式的，即运算符在操作数的后面。例如“2 3 +”等价于中缀表达式的“2 + 3”。<br>
　　请你实现一个后缀表达式的计算器。

##输入描述:
　　输入包含多组数据。<br>
　　每组数据包括两行：第一行是一个正整数n (3≤n≤50)；紧接着第二行包含n个由数值和运算符组成的列表。<br>
　　“+-*/”分别为加减乘除四则运算，其中除法为整除，即“5/3=1”。

##输出描述:
　　对应每一组数据，输出它们的运算结果。

##输入例子:
```
3
2 3 +
5
2 2 + 3 *
5
2 2 3 + *
```

##输出例子:
```
5
12
10
```
