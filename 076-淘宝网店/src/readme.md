#淘宝网店

##题目描述
　　NowCoder在淘宝上开了一家网店。他发现在月份为素数的时候，当月每天能赚1元；否则每天能赚2元。<br>
　　现在给你一段时间区间，请你帮他计算总收益有多少。

##输入描述:
　　输入包含多组数据。<br>
　　每组数据包含两个日期from和to (2000-01-01 ≤ from ≤ to ≤ 2999-12-31)。<br>
　　日期用三个正整数表示，用空格隔开：year month day。


##输出描述:
　　对应每一组数据，输出在给定的日期范围（包含开始和结束日期）内能赚多少钱。

##输入例子:
```
2000 1 1 2000 1 31
2000 2 1 2000 2 29
```

##输出例子:
```
62
29
````
