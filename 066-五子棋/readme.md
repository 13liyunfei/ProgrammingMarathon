#五子棋

##题目描述
　　NowCoder最近爱上了五子棋，现在给你一个棋局，请你帮忙判断其中有没有五子连珠（超过五颗也算）。

##输入描述:
　　输入有多组数据，每组数据为一张20x20的棋盘。<br>
　　其中黑子用“*”表示，白子用“+”表示，空白位置用“.”表示。


##输出描述:
　　如果棋盘上存在五子连珠（无论哪种颜色的棋子），输入“Yes”，否则输出“No”。

##输入例子:
```
....................
....................
....................
....................
......*.............
.......*............
........*...........
....++++.*..........
..........*.........
....................
....................
....................
....................
....................
....................
....................
....................
....................
....................
....................
....................
....................
....................
.......*............
......+*+++.........
.......*............
.......*............
....................
....................
....................
....................
....................
....................
....................
....................
....................
....................
....................
....................
....................
```

##输出例子:
```
Yes
No
```
